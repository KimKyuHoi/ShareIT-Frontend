### Usage

1. Git Clone

```
$ git clone https://github.com/KimKyuHoi/ShareIT-Frontend.git
```
2. Install package

```cmd
$ yarn

# or

$ npm install
```
3. Start

```cmd
$ yarn start

# or

$ npm run start
```
